Coupon Creation:

To create a coupon, you must fill out all fields

Note !

The title field will appear on the image in the customer display pen

The Start Date field must be at least from today.

The Expiration Date field must be at least 5 days from the start date.

If the start date is over the 24th of the month, the expiration date will be 6 next month.

My coupons:

The company's coupons will be displayed. They can be filtered by title and ID

Coupon Search by ID:

Must enter into the search field an exact ID corresponding to the coupon belonging to the company.

Get coupon up to expiration date:

Filter coupons up to the date entered in the field

Will return all coupons if there are according to the date entered.

Update and delete coupon:

Update coupon information You can update the expiration date and coupon price of the coupon

Please note

that the update date can only be five days from the start or day date if the start date is earlier than today

it can only be five days from now.

Note !

Deleting a coupon will lose the coupon from the entire system, including from the customer.
